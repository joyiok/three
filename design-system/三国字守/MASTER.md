# 三国字守 · 设计系统（MASTER · 单一真相源）

> 由 UI UX Pro Max skill 生成基线，结合水墨三国塔防主题定制。后续所有 UI 工作以此为准。

## 主题
三国题材文字塔防。纸面水墨风（E-Ink/Paper + Anti-Polish/Raw Aesthetic 混合）：米黄宣纸底、墨色汉字、朱砂红强调、金色星级。竖屏移动优先，桌面 max-width 480px 居中。无图片素材，Canvas 程序化绘制 + DOM UI。

## 字体

- 标题/单位字：`"Kaiti SC","KaiTi","STKaiti","Noto Serif SC","Songti SC",serif`（楷体，凸显汉字美感）
- UI 数字/小字：同上楷体；按钮文字字间距加大 4-8px 增强仪式感
- 字号：主标题 64-88px、关名 22px、单位字 = 0.62×格宽、等级角标 0.22×格宽、HUD 12-14px

## 风格关键词
纸感纹理、墨色高对比、手绘描边（不规则 1px）、圆角 6-10px、柔和投影、无渐变光泽、朱砂点缀。避免：霓虹、玻璃拟态、深色模式、3D 阴影、AI 紫粉渐变。

## 关键效果
- 过渡 150-300ms ease-out；按压缩放 0.92-0.96；页面水墨晕开转场 0.4s
- 飘字/粒子由 Canvas 处理（墨爆、赏金上飘、刀光弧线）
- 漏怪红光 vignette、Boss 震屏、合成墨圈扩散
- 所有可点击 cursor: pointer；尊重 prefers-reduced-motion

## 布局

### 全局
```
#app 480px 居中列，flex column，dvh 100%
  ├─ .hud     顶栏（粮食/血量/关名/波次/暂停/倍速/静音）
  ├─ .stage   flex:1  Canvas 竖屏（7×10 格，宽=100%）
  └─ .dock    底栏（营帐/铲子/6槽/征兵/开波）
```

### HUD
左：粮食 pill（🍚+数）、血量 pill（♥+数，掉血抖动红光）
中：关名（楷体加粗）+ 第n波/共m波
右：暂停⏸ / 倍速 1x / 静音 🔇（30px 方角图标按钮）

### Dock
- 顶排：营帐图标（CSS 画三角顶+方体「营」字）+ 铲子槽 ⛏ + 6 个士兵槽
- 士兵槽：38px 方格，浅米底+纸边描线，楷体大字居中，忠字用金色，等级角标右上红字
- 征兵按钮：木色渐变全宽，44px，楷体"征 兵"+ 价格，字间距 4px
- 开波按钮：朱砂渐变全宽，34px，"开 波"/"开 战"，休整态显示倒计时

### 菜单页
- 顶部：副标「三国」（小字、宽字距）+ 主标「字守」（64-88px 楷体粗、墨色、带墨晕投影）
- 标语「墨守营寨 · 字御千军」（灰墨小字）
- 中部：装饰大字「山/守」（浅墨半透，作纸面印记）
- 底部：「开 始」朱砂大按钮 + 玩法提示小字

### 选关页
- 顶部：返回 ← + 「选 关」标题
- 4 张卷轴卡：浅米底、左侧朱砂竖条、楷体关名、星级★★★、波数/Boss 信息
- 锁定卡灰化 0.5 opacity + 🔒

### 结算页
- 居中：「胜 利」朱砂大字 / 「营寨失守」墨色
- 胜：★★★ 逐颗弹出动画（金色，带光晕）
- 败：大「败」字朱砂
- 「再 战」朱砂按钮 + 「下 一 关」玉绿按钮（胜利且有下一关时）+ 「选关」文字链
