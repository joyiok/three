# 三国字守

一款网页三国题材文字塔防游戏。所有单位以汉字表现：随机征兵获得「刀/枪/弓/骑/忠」士兵，拖放到草地格防守；敌人「斗/贼/兵/将/Boss」沿土路行进，走到终点扣营寨血量。同字同级士兵可合成升级。4 关战役，纸质水墨美术风，全程程序化绘制，无任何图片/音频素材。

## 玩法

- **网格**：7 列 × 10 行。`#`=土路（敌人行进）、`.`=草地（可部署）、`r`=岩石（装饰）。
- **经济**：初始粮食 15。征兵初价 10、每次 +2、上限 30。击杀赏金 +2/+4/+10/+50，每波清空奖励 +10。铲子回收返还 4×等级粮食。
- **士兵**（1 级基准；每升 1 级伤害 ×1.8、射程 +0.15 格）：
  - 刀：近战快攻，射程 1.3、攻速 1.2、伤害 10，单体。
  - 枪：直线穿刺，射程 2.5、攻速 0.8、伤害 8，伤害命中方向直线上所有敌人。
  - 弓：远程单体，射程 3.5、攻速 1.0、伤害 12，弹道箭矢。
  - 骑：范围重击，射程 2.0、攻速 0.5、伤害 25，溅射半径 0.8 格。
  - 忠：减速光环，半径 1.5，减速 35%，每级 +5%，上限 60%（多个光环取最强，不叠加）。
- **合成**：同字且同级两个士兵（背包内、已部署、或两者之间拖放）合为等级 +1 士兵。
- **索敌**：默认攻击射程内进度最靠前的敌人。
- **敌人**（HP = 基础 × (1 + 0.15×(波次-1)) × 关卡系数）：斗 30/速1.0、贼 20/速1.6、兵 80/速0.8、将 300/速0.7、Boss 关底（HP 见关卡）。
- **波次**：每关 10~12 波；一波全灭 3 秒自动开下一波（可手动「开波」）；末波出 Boss。
- **胜负**：营寨血量 10，归零判负；通关末波判胜。结算星级：剩余血量 ≥9 三星、≥5 二星、否则一星。
- **倍速/暂停/静音** 常驻 HUD；页面失焦自动暂停。

## 技术栈

- Vite 5 + TypeScript（strict）+ Vitest，零运行时依赖。
- Canvas 2D 程序化绘制纸质水墨地图与单位；Web Audio API 合成全部音效与古风 BGM（五声音阶）。
- localStorage 存档进度（`sgtd.progress`）与静音（`sgtd.muted`），不可用时降级为会话内存态。

## 模块

```
src/
  main.ts          入口、页面路由（菜单/选关/对局/结算）、GameScreen 游戏循环与拖放
  storage.ts       进度存档与静音状态
  audio.ts         Web Audio 音效与 BGM
  game/
    types.ts       共享类型
    config.ts      数值配置（兵种、敌人、经济、合成）
    levels.ts      4 关地图、路径、波次表
    state.ts       对局状态机与操作（征兵、部署、合成、回收）
    waves.ts       波次调度、敌人移动、胜负判定
    combat.ts      索敌、攻击、弹道、减速光环
    engine.ts      游戏循环、时间推进、倍速
  render/
    canvas.ts      地图与单位绘制（纸质水墨风）
    effects.ts     粒子、飘字、墨爆、震屏
  ui/
    hud.ts         顶部 HUD（粮食/血量/波次/暂停/倍速/静音）
    dock.ts        底部面板（营帐/铲子/6 背包槽/征兵/开波按钮）
    screens.ts     菜单/选关/结算页面
tests/             Vitest 单元测试（纯逻辑模块）
```

## 开发命令

```bash
npm install      # 安装依赖
npm run dev      # 启动开发服务器
npm run build    # 类型检查 + 生产构建
npm run preview  # 预览构建产物
npm test         # 运行单元测试（vitest）
npx tsc --noEmit # 仅类型检查
```

## 测试

Vitest 覆盖纯逻辑模块（61 用例）：配置数值、关卡数据、对局状态、波次调度、战斗、引擎整合、进度存档、平衡自洽性（空场必败、强力阵容能通黄巾）。渲染与拖放在浏览器人工验证。

## 平台

移动端竖屏优先，桌面端 `max-width: 480px` 居中。支持鼠标与触摸（Pointer Events）。
