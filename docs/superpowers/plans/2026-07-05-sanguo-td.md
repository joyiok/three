# 三国文字塔防 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 按 `docs/superpowers/specs/2026-07-05-sanguo-td-design.md` 实现一款可在手机/桌面浏览器游玩的三国文字塔防（4 关战役、随机征兵、拖放合成、程序化水墨美术与 Web Audio 音效）。

**Architecture:** 纯逻辑核心（state/waves/combat/engine，无 DOM 依赖，Vitest 覆盖）+ Canvas 渲染层 + DOM UI 层（HUD、底部面板、页面路由）。所有帧事件通过 `GameState.events` 队列传给渲染与音效。

**Tech Stack:** Vite 5 + TypeScript（strict）+ Vitest，零运行时依赖，Canvas 2D，Web Audio API。

## Global Constraints

- 全部 UI 文案为简体中文；单位字体栈 `"Kaiti SC","KaiTi","STKaiti","Noto Serif SC",serif`。
- 无任何运行时 npm 依赖；无图片/音频素材，全部程序化生成。
- 竖屏布局，桌面上游戏列 `max-width: 480px` 居中。
- localStorage 键：进度 `sgtd.progress`（JSON `{stars: Record<levelId, number>}`），静音 `sgtd.muted`。
- 网格 7 列 × 10 行；坐标 `{x:列, y:行}`，左上角为 (0,0)。
- 数值一律取自设计文档 §3（本计划 Task 2 已内联全部数值）。

export interface Soldier { id: number; kind: SoldierKind; level: number; cell: Vec | null; cooldown: number }
export interface Enemy { id: number; kind: EnemyKind; word: string; hp: number; maxHp: number;
  speed: number; slow: number; pathIndex: number; progress: number; bounty: number; damage: number }
export interface Projectile { id: number; x: number; y: number; targetId: number; speed: number; damage: number }

export type GameEvent =
  | { t: 'recruit' } | { t: 'deploy' } | { t: 'merge'; cell: Vec; level: number }
  | { t: 'sell' } | { t: 'shoot'; kind: SoldierKind; from: Vec }
  | { t: 'hit'; x: number; y: number; damage: number } | { t: 'kill'; x: number; y: number; bounty: number }
  | { t: 'leak'; damage: number } | { t: 'waveStart'; wave: number }
  | { t: 'boss'; word: string } | { t: 'won' } | { t: 'lost' };
export interface GameState { levelId: string; food: number; recruitCost: number; baseHp: number;
